Mandelbrot Set - Python

Github

This project was one of the first things I tried in Python. It was a quick project with the goal of drawing the Mandelbrot set on a two dimensional canvas and then generating a png from this canvas. I drew inspiration from the cool appearance of the set itself and I figured it'd be good practice while also producing interesting results.

My first version was essentially just a draft - something which technically worked but had zero flexibility and only functioned at a bare minimum.

The second version is much cleaner, allowing for precise control of the zoom and location of the drawn section of the mandelbrot set.

I don't have plans to revisit this program as I feel it's done a good enough job at familiarizing me with Python. If I were to upgrade it I would focus on implementing a color gradient to represent the rate at which certain values escape the set. This would allow for more vivid imagery as many of the colorful images people associate with the Mandelbrot set are actually consist of values entirely outside of the set.
